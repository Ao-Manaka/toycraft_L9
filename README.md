### 3. 原版 Minecraft 修改内容

* 爆炸保护、火焰蔓延关闭，死亡掉落开启

* 全部的黄金合成表被禁止（包括金装备和金武器、工具）。



### 4. 核心 mod 修改内容

#### (101) 枪械模组

* 枪械工作台的合成已被禁止，所有枪械子弹及下文中的相关物品均需在出生点的枪械店进行交易。

* 可选枪械：手枪、AK47、冲锋枪、霰弹、M4A1、AWM（狙击）、榴弹发射器、加特林机关枪、迫击炮

* 
#### (102) 失落城市

* 建筑物内奖励箱内容：基岩、恶魂之泪、钻石、附魔之瓶、金块、烈焰棒、末影珍珠、金锭

* 建筑物内奖励箱开出是唯一获取基岩的方式，基岩将作为制约各核心 mod 发展的关键物品。

#### (103) 急救模组

* 取消了吗啡和绷带的制作配方（使用金锭兑换），修改了创可贴的制作配方（腐肉制作）


### 5. 其余 mod 修改内容

#### (201) 载具模组

* 取消了所有发动机的制作配方，所有的发动机均需要通过使用金块换取。
